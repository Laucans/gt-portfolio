# Laurent Canis — Portfolio

Portfolio statique de Laurent Canis, Engineering Leader & MBA HEC Montréal.

## Structure

```
src/         → Sources du site (HTML, CSS, assets)
docs/        → Documents source (CV, LinkedIn export, assets visuels)
tests/       → Tests
scripts/     → Scripts de build et utilitaires
```

## Commandes

```bash
npm start    # Servir le site en local (http://localhost:3000)
npm run build  # Copier src/ → dist/ pour déploiement
npm test     # Valider le HTML
npm run lint # Linter le HTML
```

## Stack

- HTML5 sémantique, CSS vanilla
- Site 100% statique, aucun backend
- Mobile-first, dark mode, scroll animations
